# Temel Kavramlar (Core Concepts) 🏮

Bu bölüm, Sun Tzu stratejisinin üç ana direğini derinlemesine inceler.

## 1. Tao (Yol / Hizalanma)
Tao, bir organizasyonun "neden" var olduğudur. Ekip içindeki tam uyumdur. Modern mühendislikte bu, **Şirket Kültürü** ve **Misyon Hizalanması**'dır.

## 2. Qi (Dolaylı Güç / Yaratıcılık)
Qi, beklenmeyendir. Standart süreçlerin ötesine geçen yaratıcı enerjidir.

## 3. Shi (Momentum / Enerji)
Shi, doğru zamanda serbest bırakılan potansiyel güçtür. Bir barajın kapaklarının açılması gibi.
